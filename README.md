## ❓ Часто задаваемые вопросы

**❓ Что нужно для работы?**

-   Windows 7/8/10/11
    
-   Установленный League of Legends
    
-   Запущенный клиент League of Legends
    

**❓ Это безопасно?**  
✅ Да! Скрипт работает только с локальным API твоего клиента League of Legends. Никакие данные не отправляются в интернет.

**❓ Почему не видно названий скинов, только ID?**  
API инвентаря возвращает только ID скинов. Это база для дальнейшей работы — можешь использовать эти ID для получения полной информации через другие эндпоинты.

**❓ Сколько скинов я найду?**  
Столько, сколько у тебя куплено. Скрипт показывает только те скины, которые реально есть в твоем инвентаре.
